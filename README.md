# Rekam Medis Klinik 
### Project Based Learning (PBL) - Sistem Basis Data

Repository ini berisi seluruh pengembangan basis data untuk proyek pembelajaran berbasis proyek (PBL). Fokus utama proyek ini adalah **perancangan, normalisasi, dan implementasi basis data relasional** untuk manajemen rekam medis klinik, mulai dari analisis kebutuhan hingga pengujian query.

> ⚠️ **Catatan:** Sesuai ketentuan PBL, proyek ini berfokus pada integritas data, relasi tabel, normalisasi, dan query SQL.
