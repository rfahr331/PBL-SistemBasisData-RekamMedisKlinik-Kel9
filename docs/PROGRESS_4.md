# PROGRES 4: PRESTASI DAN LAPORAN AKHIR

**Mata Kuliah:** Sistem Basis Data  
**Studi Kasus:** Sistem Rekam Medis Klinik  
**Kelompok:** 9

---

## Deskripsi

Progres 4 merupakan tahap akhir dari pengembangan **Sistem Rekam Medis Klinik**. Pada tahap ini dilakukan penyempurnaan implementasi basis data, pengujian seluruh fitur yang telah dibuat, serta penyusunan laporan akhir proyek. Seluruh komponen database telah berhasil diimplementasikan menggunakan MySQL, mulai dari pembuatan tabel, relasi antar tabel, pengisian data, hingga pengujian query dan trigger.

---

## Capaian Proyek

Pada Progres 4, kelompok berhasil menyelesaikan beberapa pencapaian berikut:

- Merancang dan mengimplementasikan database Sistem Rekam Medis Klinik.
- Membuat relasi antar tabel menggunakan **Primary Key** dan **Foreign Key**.
- Mengimplementasikan operasi **DDL (Data Definition Language)**.
- Mengimplementasikan operasi **DML (Data Manipulation Language)**.
- Membuat dan menguji berbagai query SQL.
- Mengimplementasikan **Trigger** untuk otomatisasi proses pada database.
- Melakukan pengujian seluruh fitur database.
- Menyusun laporan akhir proyek.
